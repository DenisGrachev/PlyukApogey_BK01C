# PlyukApogey_BK01C

Source code for Plyuk game for soviet PC Apogey-BK01C as is

## Quick Start

Grab **sjasmplus** from **z00m128** https://github.com/z00m128/sjasmplus, download sources, put **sjasmplus.exe** in the root folder and compile **main.asm**
